Welcome to our game, Jump King x Doodle Jump! 

In order to run our program, make sure the following python libraries are downloaded, and your version of python is 3.12 or greater.

You can run the following commands:
pip install pygame
pip install typing

The other libraries used (sys, time, math, and random) come with Python.

run the following command to get started:
python main.py

NOTE: This game creates a CSV which saves your highscore. If you want to reset your highscore, just delete the file.

Have fun playing!
